1. JS内置类型

最新的ECMAScript标准定义了 8 种数据类型：

  • 7种原始类型（值本身无法被改变）
    • 布尔类型：只有两个值 true 和 false
    • NULL： 只有一个值 null
    • Undefined：没有被赋值的变量会有一个默认值undefined
    • String：文本数据
    • Number：限制 -（2^53 -1）~ （2^53 - 1）
    • BigInt：可以安全地存储和操作大整数，通过在整数末尾附加n或调用构造函数来创建的
    • Symbol：符号类型是唯一的并且是不可修改的，并且可以作为Object的key值
  • Object（对象指内存中可以被标识符引用的一块区域）

1. Number和BigInt

Number是有范围限制的，但是为了安全的存储和操作大整数，甚至可以超过数字的安全整数限制，就有了BigInt。

jquery ==> DataTable插件报错：Uncaught TypeError: Cannot read property ‘style’ of undefined

解决：

table 中定义的列和aoColumns数据表的项数量对不上，注意定义的列数

Element UI select样式无法修改

解决：

1. 将el-select加上 :popper-append-to-body="false"，然后添加一个新的class
如下：
  <el-select v-model="optionValue1" popper-class="select-down" :popper-append-to-body="false" @change="optionChange1" :disabled="!netState" placeholder="请选择" class="select-option" size="">
                    <el-option
                      v-for="item in options1"
                      :key="item.type"
                      :label="item.name"
                      :disabled="item.disabled"
                      :value="item.type">
                    </el-option>
                  </el-select>

2. 在对应的结构给上样式，如果不行就加!important           
      /deep/.select-down .el-select-dropdown__item.is-disabled:hover {
        background-color: #0E2B4A !important;
      }
